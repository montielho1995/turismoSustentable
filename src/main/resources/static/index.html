¿Quiénes Somos?

Somos una plataforma digital que facilita el alquiler de bicicletas inteligentes, diseñadas tanto para turistas como para residentes de Río Grande.

Nuestro principal objetivo es promover un modelo de turismo sustentable ofreciendo una alternativa flexible, económica y saludable para explorar cada rincón de la ciudad. Buscamos aliviar el tráfico, reducir la huella de carbono y poner en valor el patrimonio local.
