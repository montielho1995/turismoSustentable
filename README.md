# Turismo Sustentable

Proyecto de desarrollo web para la gestión de alquiler de bicicletas en Rio Grande, Tierra del Fuego.
Desarrollado como parte de la Tecnicatura Superior en Desarrollo de Software.

## Tecnologías utilizadas
- Java
- Spring Boot
- SQL (Base de datos relacional)
- Figma (Diseño UI/UX)

## Descripción
Sistema enfocado en la gestión de alquileres, estado de bicicletas y autenticación de usuarios.
